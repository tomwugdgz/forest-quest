🌲 森林冒险家园 — 家务轮盘游戏设计文档
一、游戏概述
游戏名称: 森林冒险家园 (Forest Quest)

核心玩法: 小朋友每天通过"森林之神的转盘"抽取守护角色卡，完成对应家务任务，积累经验值和金币，提升等级，解锁成就。

目标年龄: 3-12岁（分龄设计）

世界观: 小朋友是森林小镇的小冒险家，每天由森林之神赐予角色，通过完成任务守护自己的家园。

设计理念:

主人翁精神：让小朋友觉得是"森林家园的守护者"
分龄设计：3-6岁和7-12岁有完全不同的角色体系和任务难度
可玩性：角色卡系统+轮盘抽奖带来持续惊喜感
成就感：经验值成长+成就系统+图鉴收集可视化进步
公平性：多角色卡随机抽取，保证家务分配均衡，减少争抢
二、年龄分层体系
🌱 第一阶段：小小冒险家（3-6岁 · 学龄前）

6.2 多孩家庭公平机制
问题：两个孩子可能抽到"简单任务"和"困难任务"，引发不公平感。

解决方案：

难度等价：N卡1个 = R卡1个 = 同等级别（经验值不同但完成都获得认可）
交换机制：两个孩子可以协商交换角色卡（增加互动和沟通）
合作任务：每周出现1次合作卡，需要两人一起完成，获得双倍奖励
随机性公平：不是按能力分配，而是纯随机，减少"为什么他总是做简单的"的抱怨

十、游戏设计原则总结
即时反馈：完成任务立刻获得经验值和金币
可见成长：等级、成就、图鉴可视化进步
分龄适配：3-6岁和7-12岁有截然不同的角色体系
持续激励：连续签到、成就、奖励商店形成激励循环
趣味性：角色扮演、轮盘抽奖、卡片收集
公平和谐：随机抽取+交换机制+合作任务，减少争抢
主人翁精神：让小朋友觉得是"守护家园"，不是"被逼做家务"
