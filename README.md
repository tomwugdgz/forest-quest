# 🌲 森林冒险家园 (Forest Quest)

> 一款专为 3-12 岁小朋友设计的**家务角色扮演轮盘**微信小程序，让孩子在游戏中主动承担家务，培养责任感和生活能力。

## ✨ 核心功能

| 模块 | 说明 |
|------|------|
| 🎡 **转盘抽奖** | Canvas 动画转盘，每天抽取当日"角色任务" |
| 📋 **今日任务** | 查看已分配任务，完成打卡，追踪进度 |
| 🃏 **角色图鉴** | 收集全部角色卡（N/R/SR 稀有度），按年龄筛选 |
| 🏪 **奖励商店** | 金币兑换真实奖励（额外游戏时间、零食自选等） |
| 🏆 **成就系统** | 18 种成就，持续激励 |
| 🔐 **家长中心** | 密码保护，管理年龄段、任务数、金币倍率、数据重置 |

## 🎮 年龄分层

### 🌱 小小冒险家（3-6岁）
- 12 张角色卡，全部为协助类家务
- 拟人化任务描述（"给桌子洗脸"、"碗筷回家"）
- 每次抽取 **1 张卡**

### 🌳 正式冒险家（7-12岁）
- 14 张角色卡（含 N/R/SR），可独立完成
- 开始学习使用家电
- 每次抽取 **1-2 张卡**

### 👑 森林领袖（9-12岁进阶）
- 6 张角色卡，以 SR 为主
- 参与家庭事务决策（预算规划、活动组织）
- 培养统筹能力和领导力

## 🚀 快速开始

### 微信小程序

1. 打开**微信开发者工具**
2. 导入项目 — 选择 `forest-quest/` 目录
3. 点击编译预览

### 网页预览版

直接用浏览器打开 `index.html`，完整交互体验。

## ⚙️ 家长设置

默认家长密码：`1234`

可配置项：
- **孩子切换** — 支持两个孩子独立数据
- **年龄段** — 3-6岁 / 7-12岁 / 进阶
- **每日任务数** — 1-5 张任务卡
- **金币倍率** — 标准(1x) / 鼓励(1.5x) / 激励(2x)
- **修改密码**
- **重置数据** — 仅重置今日或全部

## 🎨 设计亮点

- **N卡 60%** — 基础角色，鼓励日常习惯
- **R卡 30%** — 稀有角色，中等挑战
- **SR卡 10%** — 传说角色，高难度高奖励
- 金币 + 经验值双轨奖励
- 连续天数奖励（7/14/30/60/90天阶梯）
- 成就自动解锁 + 金币奖励

## 🛠️ 技术栈

- 微信小程序原生框架（WXML + WXSS + JS）
- Canvas 2D 转盘动画
- wx.setStorageSync 本地持久化
- HTML5 Canvas 网页预览
