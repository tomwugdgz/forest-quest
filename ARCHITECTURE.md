## 四、AI 智能对话模块设计 🤖

### 4.1 AI 每日趣味知识机制

**设计目标**: 每天由 AI 扮演"森林长者"角色，用适合小朋友的语言推送一条趣味知识 + 对应的角色任务建议。

**交互流程**:
```
打开首页 → AI 主动推送今日知识
    ↓
小朋友/家长阅读知识
    ↓
AI 根据知识推荐一个任务角色
    ↓
点击「抽取今日角色」→ 进入轮盘
    ↓
（对话中也可随时提问"森林长者"）
```

**每日知识生成策略 (无需付费 API 版本)**:

```javascript
// data/ai-tips.js - 本地趣味知识库

    suggestedRole: '浇水小园丁',
    suggestedCharId: 'L10',
```

### 4.2 AI 对话能力设计

| 功能 | 描述 | 实现方式 |
|------|------|---------|
| 每日知识推送 | 首页自动展示一条趣味知识 | 本地预置 365 条 |
| 智能任务推荐 | 根据知识推荐对应角色卡 | 知识-角色映射 |
| 自由对话 | 小朋友可以问问题给"森林长者" | 方案1: 关键词匹配 / 方案2: API 接入 |
| 鼓励语句 | 完成任务后 AI 给出随机鼓励 | 本地鼓励语料库 |
| 天气适配 | 根据天气推荐不同任务 | 天气 API + 规则匹配 |

## 五、快速上线策略

## 六、技术选型总结

| 模块 | 技术 | 理由 |
|------|------|------|
| 框架 | 原生小程序 | 性能最优，无需构建 |
| 数据 | wx.setStorageSync | 简单场景够用 |
| AI | 本地语料 → API | MVP 先本地，后续接入 |
| 部署 | 微信开发者工具 | 官方工具链 |
| 分析 | 微信统计后台 | 免费 |
